# スプラトゥーン3 ブキルーレット

スプラトゥーン3に登場する全172種類の武器からランダムに1つを選択するWebアプリケーションです。

## 機能

- ボタンをクリックするだけでランダムに武器を選択
- 選択された武器の画像と名前を表示
- Twitterでの結果共有

## デモ

「スタート」ボタンをクリックすると、武器がランダムに選ばれて表示されます。

## 技術スタック

**フロントエンド:**
- HTML5
- CSS3
- Vanilla JavaScript

**武器データ更新スクリプト:**
- Bun (TypeScript ランタイム)
- node-html-parser

## プロジェクト構造

```
splatoon3_random-weapon/
├── index.html           # メインHTMLファイル
├── css/
│   ├── reset.css        # CSSリセット
│   └── style.css        # スタイルシート
├── js/
│   ├── handler.js       # イベントハンドリング・DOM操作
│   └── weapons.js       # 武器データ（172種類）
├── images/              # 武器アイコン画像
└── scripts/             # 武器データ更新用スクリプト
    ├── package.json
    ├── .env.sample
    ├── downloader/
    │   └── v2/index.ts  # スクレイピングスクリプト
    └── upgrade.sh       # 更新実行スクリプト
```

## 使い方

### Webアプリとして使用

`index.html` をブラウザで開くだけで使用できます。

### 武器データの更新

新しい武器が追加された場合、以下の手順でデータを更新できます。

1. `scripts` ディレクトリに移動
   ```bash
   cd scripts
   ```

2. 依存関係をインストール
   ```bash
   bun install
   ```

3. 環境変数を設定
   ```bash
   cp .env.sample .env
   # .env ファイルを編集して SITE_URL を設定
   ```

4. 更新スクリプトを実行
   ```bash
   ./upgrade.sh
   ```

これにより、武器データと画像が自動的に更新されます。
